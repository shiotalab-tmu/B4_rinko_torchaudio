# Discontinued Documents（旧版・廃止済み）

ここに置いてあるのは，この勉強会教材の **過去の検討段階で書いたドキュメント** です．
**いずれも現行方針ではありません．** 当時どう考えていたか（思考の記録）を残す目的で保管しています．

参照する際は「これは古い・ディスコン」という前提で読んでください．

## 中身
- `plan.md` — 最初の実装計画．穴埋め notebook ＋ 解答ブランチ参照を前提にし，「フレームワーク内部の理解」を主目的に置いていた版．
- `curriculum.md` — 各回内容のまとめ．上記の旧方針ベース．

## なぜ廃止したか（要点）
- 目的を「内部理解」から **「PyTorch で動く学習・評価スクリプトを自分で書けるようになる」** に置き直した（題材の音声認識は手段）．
- 受講者は「ゼロから作る」で理論は履修済み・実装経験が乏しい，という前提に合わせた．
- 「解答を公開する＋穴埋め」だと丸写しで通過できてしまう穴があったため，**解答は公開しない＋発表で動作確認の例題をその場で実演**する方式に変更．

現行の確定方針は別管理（リポジトリ刷新時に新しいドキュメントとして作成予定）．
